# Spectra - AI Agent & Workflow System

Spectra is a system of specialized AI agents and workflow rules. Use it with Claude, Cursor, or any AI tool.

## Quick Start

1. **Install via npm:**
   ```bash
   npm install spectra
   ```

2. **Tell your AI tool:**
   ```
   Please act as the workflow defined in: ./.spectra/rules/workflow_v4.md
   ```

3. **Describe your task** - The AI will use specialized agents to help.

## Available Agents

- 🏗️ backend-engineer - APIs, databases, server logic
- 🎨 frontend-engineer - UI, components, client code
- 📊 database-engineer - Schema, migrations, queries
- ✅ code-reviewer - Code quality, best practices
- 🧪 qa-debugger - Testing, debugging, validation
- 📋 product-manager - Requirements, scope
- 🧠 domain-expert - Business logic validation
